# Algorithms and Data Structures

This repository contains my solutions to data structures and algorithm problems. They are primarily drawn from LeetCode and are organized by data structure and solution pattern.

The goal of this repository is not to collect as many solutions as possible, but to practice and document clear efficient approaches to common data structure and algorithm problems while emphasizing on growth, correctness, readability, and trade-offs.
